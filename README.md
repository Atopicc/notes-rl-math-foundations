# 强化学习数学原理学习笔记

[![Python](https://img.shields.io/badge/Python-3.9%2B-blue)](https://www.python.org/)
[![PyTorch](https://img.shields.io/badge/PyTorch-1.9%2B-red)](https://pytorch.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange)](https://jupyter.org/)

这是学习西湖大学赵世钰老师《强化学习的数学原理》一书的学习笔记和代码实现。

## 📚 项目简介

本项目包含了《强化学习的数学原理》一书的学习笔记和配套代码实现，旨在通过理论结合实践的方式深入理解强化学习的数学基础。

## 🛠️ 技术栈

- **编程语言**: Python 3.9+
- **深度学习框架**: PyTorch
- **数值计算**: NumPy
- **笔记环境**: Jupyter Notebook
- **可视化**: Matplotlib

## 🚀 快速开始

### 环境配置

1. 克隆项目到本地：
```bash
git clone https://github.com/your-username/notes-rl-math-foundations.git
cd notes-rl-math-foundations
```

2. 创建并激活虚拟环境：
```bash
conda create -n rl-math python=3.9
conda activate rl-math
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

### 运行示例

启动 Jupyter Notebook：
```bash
jupyter notebook
```

然后在浏览器中打开相应的章节笔记文件即可开始学习。
